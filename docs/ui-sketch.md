# UI Wireframe Sketches

סקיצה טקסטואלית למסכי המערכת העיקריים. לכל מסך מצוין היעד העיקרי, רכיבי הליבה והערות חוויית משתמש.

## 1. מסך התחברות (Login)

```
+--------------------------------------------------+
|                    לוגו חברה                     |
|--------------------------------------------------|
|   ברוך הבא!                                      |
|                                                  |
|   [ אימייל......................... ]            |
|   [ סיסמה......................... ]             |
|                                                  |
|   (כניסה)                                        |
|   שכחת סיסמה?                                    |
|                                                  |
|   אין לך משתמש מערכת? פנה למנהל הארגון          |
+--------------------------------------------------+
```

**יעד**: אימות משתמשי מערכת.

**רכיבים**:
- שדות אימייל/סיסמה עם ולידציה בזמן אמת.
- לינק "שכחת סיסמה" המוביל לתהליך שחזור.
- הודעת שגיאה צפה (toast) במקרה של התחברות כושלת.

## 2. דשבורד מנהל-על (Super Admin Dashboard)

```
+----------------------+---------------------------+
|      תפריט צד        |  כותרת: ברוך הבא, מנהל   |
|----------------------|---------------------------|
|  לוח מחוונים        |  סטטיסטיקות כלליות:      |
|  ארגונים             |  - # ארגונים פעילים      |
|  קורסים              |  - # מנויים פעילים       |
|  דיווחים             |  גרף קווי: מנויים/זמן     |
|  משתמשים             |  כרטיסים:                |
|                      |  [ארגון A] [ארגון B] ...  |
+----------------------+---------------------------+
```

**יעד**: תמונת מצב גבוהה למנהל העל לגבי כלל המערכת.

**רכיבים**:
- כרטיסי KPI (ארגונים פעילים, מנויים, קורסים משויכים).
- גרף מגמות (מנויים חדשים בחודש האחרון).
- רשימת פעולות אחרונות (audit trail קצר).

## 3. רשימת ארגונים + דף פרטי ארגון

### רשימת ארגונים
```
+--------------------------------------------------+
| כפתור "הוסף ארגון"                               |
|--------------------------------------------------|
| חיפוש: [..........................]              |
| פילטרים: סטטוס ▼  תאריך יצירה ▼                |
|--------------------------------------------------|
| שם ארגון | #קבוצות | #מנויים | עוד פעולות ▼      |
|--------------------------------------------------|
| ארגון A  |    4    |   120   | ערוך | מחק        |
| ארגון B  |    2    |    45   | ערוך | מחק        |
+--------------------------------------------------+
```

### פרטי ארגון
```
+--------------------------------------------------+
| < חזרה לרשימה                                      |
|----------------------------------------------------|
| כותרת: ארגון A                                     |
| ---------------------------------------------------|
| טאבים: מידע כללי | קורסים | קבוצות | משתמשי מערכת |
|----------------------------------------------------|
| (טאב מידע כללי)                                    |
| שם ארגון: [ארגון A________]                        |
| תאור:    [__________________]                      |
| סטטוס:   פעיל ▼                                    |
| כפתורים: (שמור) (בטל)                             |
+----------------------------------------------------+
```

**יעד**: ניהול ארגונים, שיוך קורסים, קבוצות ומשתמשי מערכת.

**רכיבים**:
- כפתור פעולה להוספה/עריכה/מחיקה.
- טאב קורסים מציג טבלה של הקורסים המשויכים + כפתור "שיוך קורס".
- טאב קבוצות מציג טבלה + כפתור "הוסף קבוצה".
- טאב משתמשי מערכת מציג טבלה עם תפקידי גישה (Admin, Viewer).

## 4. מסך שיוך קורסים לארגון

```
+----------------------------------------------+
| כותרת: שיוך קורסים לארגון A                  |
|----------------------------------------------|
| תיבת חיפוש: [..................]             |
|----------------------------------------------|
| [ ] קורס 1  | משך: 8 שעות | מחיר ברירת מחדל |
| [x] קורס 2  | משך: 4 שעות | מחיר: [400]     |
| [ ] קורס 3  | משך: 6 שעות | מחיר ברירת מחדל |
|----------------------------------------------|
| (שמור שיוכים) (בטל)                         |
+----------------------------------------------+
```

**יעד**: לאפשר למנהל-על לבחור קורסים זמינים לארגון ולהגדיר התאמות (מחיר, תוקף).

**רכיבים**:
- טבלת קורסים עם תיבת בחירה.
- שדות התאמה נפתחים רק כשהקורס מסומן.
- אפשרות לשמור או לבטל.

## 5. מסך מנהל ארגון – דשבורד

```
+----------------------+---------------------------+
|      תפריט צד        |  כותרת: ארגון A          |
|----------------------|---------------------------|
|  דשבורד              |  כרטיסי סטטוס:           |
|  קורסים זמינים       |  - מנויים פעילים         |
|  מנויים               |  - קורסים פעילים         |
|  קבוצות              |  - סיום תוקף קרוב        |
|  משתמשי מערכת        |  רשימת פעולות אחרונות   |
+----------------------+---------------------------+
```

**יעד**: לתת למנהל ארגון תמונת מצב ברמת הארגון בלבד.

**רכיבים**:
- פילוחים של מנויים לפי קבוצות.
- כפתור קריאה לפעולה "הוסף מנוי".

## 6. מסך ניהול מנויים בארגון

### רשימת מנויים
```
+--------------------------------------------------+
| כפתור "הוסף מנוי"                               |
|--------------------------------------------------|
| חיפוש: [..................]  פילטר סטטוס ▼      |
|--------------------------------------------------|
| שם מלא | אימייל | קבוצות | סטטוס | תוקף | עוד ▼ |
|--------------------------------------------------|
| רות כהן | r@co.il | קבוצה 1 | פעיל | 01/01/24 |
+--------------------------------------------------+
```

### טופס מנוי
```
+--------------------------------------------------+
| כותרת: הוסף/ערוך מנוי                            |
|--------------------------------------------------|
| פרטי זיהוי:                                      |
|   שם פרטי:  [_______]                            |
|   שם משפחה: [_______]                            |
|   אימייל:    [_______] (ולידציה ייחודיות)       |
|   טלפון:     [_______]                           |
| קבוצות:  ▼ בחירת קבוצה (multi-select)           |
|--------------------------------------------------|
| הרשאות קורסים:                                   |
|  [ ] קורס 1 | תוקף: [מתאריך] - [עד תאריך]       |
|  [x] קורס 2 | תוקף: [01/01/24] - [01/04/24]      |
|--------------------------------------------------|
| פרטי תשלום (אופציונלי):                         |
|  [ערוך פרטי חיוב]                                |
|--------------------------------------------------|
| (שמור) (בטל)                                     |
+--------------------------------------------------+
```

**יעד**: יצירה וניהול של מנויי קצה ברמת הארגון.

**רכיבים**:
- טבלה עם אפשרות ייצוא (CSV/Excel).
- בטופס: לוגיקת תוקפים (בחירת תאריך התחלה/סיום), תיוג סטטוס (Active/Expired).

## 7. מסך מנוי – פרטי מנוי

```
+--------------------------------------------------+
| < חזרה לרשימה                                      |
|----------------------------------------------------|
| כרטיס מנוי:                                        |
| שם: רות כהן                                        |
| אימייל: r@co.il                                    |
| טלפון: 050-1234567                                 |
| קבוצות: קבוצה 1, קבוצה 2                          |
| סטטוס: פעיל                                        |
|----------------------------------------------------|
| קורסים פעילים:                                     |
|  - קורס 2 | 01/01/24 - 01/04/24 | הארך ▼           |
|  - קורס 5 | 15/02/24 - 15/05/24 | ביטול ▼          |
|----------------------------------------------------|
| הערות פנימיות: [______________________________]    |
| טבלת היסטוריית פעולות                             |
+----------------------------------------------------+
```

**יעד**: מתן פירוט על מנוי יחיד ופעולות תחזוקה (הארכה, ביטול, השהיה).

**רכיבים**:
- לוח פעולות (Activity log) עם פעולות אחרונות.
- כפתור "שליחת אימייל".

## 8. מסך ניהול קורסים (ברמת המערכת)

```
+--------------------------------------------------+
| כפתור "הוסף קורס"                               |
|--------------------------------------------------|
| חיפוש: [..................]                      |
|--------------------------------------------------|
| שם קורס | מחיר בסיסי | משך | # ארגונים | עוד ▼   |
|--------------------------------------------------|
| קורס 1  | 350 ₪       | 8ש  | 12         | ערוך    |
| קורס 2  | 500 ₪       | 12ש | 8          | ערוך    |
+--------------------------------------------------+
```

**יעד**: ניהול ספריית הקורסים הכוללת.

**רכיבים**:
- טופס עריכת קורס עם שם, תאור, משך, מחיר ברירת מחדל, קטגוריות.
- חיווי על מספר ארגונים שמשתמשים בקורס.

## 9. מסך דיווחים (Reports)

```
+--------------------------------------------------+
| כותרת: דיווחים                                    |
|--------------------------------------------------|
| בחר ארגון: ▼                                      |
| טווח תאריכים: [מתאריך] - [עד תאריך]             |
|--------------------------------------------------|
| גרף עמודות: מנויים חדשים לפי חודש                |
| גרף עוגה: פילוח קורסים                            |
| טבלה:                                            |
|  מנוי | קורס | תאריך התחלה | תאריך סיום | סטטוס |
|--------------------------------------------------|
| כפתור "ייצוא ל-CSV"                              |
+--------------------------------------------------+
```

**יעד**: מתן נתונים עסקיים למנהל העל ולמנהלי ארגון.

**רכיבים**:
- פילטרים מתקדמים (סטטוס מנוי, קורס ספציפי).
- אפשרות שמירה כתבנית דוח.

## 10. מסך הגדרות משתמש מערכת

```
+--------------------------------------------------+
| כותרת: הפרופיל שלי                               |
|--------------------------------------------------|
| פרטים אישיים:                                    |
|  שם: [________]                                   |
|  אימייל: user@org.com (לא ניתן לעריכה)          |
|  טלפון: [________]                                |
| שינוי סיסמה: [סיסמה נוכחית] [חדשה] [אימות]      |
| התראות: (x) קבל התראות אימייל                    |
|         [ ] קבל התראות SMS                       |
|--------------------------------------------------|
| (שמור שינויים)                                   |
+--------------------------------------------------+
```

**יעד**: לאפשר למשתמשי המערכת לעדכן פרטים אישיים ולהגדיר העדפות.

**רכיבים**:
- ולידציית סיסמה חזקה.
- הודעת הצלחה/שגיאה ברורה.

---

### זרימת ניווט מוצעת
1. התחברות → דשבורד (Super Admin או מנהל ארגון בהתאם להרשאה).
2. מתוך הדשבורד: קפיצה לרשימת ארגונים / קורסים / מנויים.
3. בתוך ארגון: טאבים עבור קורסים, קבוצות, משתמשי מערכת ומנויים.
4. כל טבלה מספקת פעולות מהירות (Action Menu) ואפשרות צפייה בפרט מנוי/קורס במסך ייעודי.

### דגשים לחוויית משתמש
- ממשק דו-לשוני: תמיכה בעברית ובאנגלית (RTL/LTR).
- שימוש ב-Design System (כגון Material UI) להאחדה וקלות פיתוח.
- הצגת הודעות מערכת באמצעות toast/notification center.
- זמינות על גבי מובייל (responsive) – טבלאות רספונסיביות עם "Cards" במובייל.
